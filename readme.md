## Tailwind Example

### Installation

Install node modules

```
npm install
```

Open terminal 1 and run

```
npx tailwindcss -i ./src/input.css -o ./public/output.css --watch
```

Open terminal 2 and run

```
serve
```

Open browser and type url address

```
http://localhost:3000/
```
